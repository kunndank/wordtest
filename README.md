# README

This application executes the uploaded dictionary file and extracts unique four-letter sequences from words present in it. Creates seperate files for the unique sequence and the word it's present in.


Ruby version used is "ruby-3.1.0"
Rails version used is "rail-7.2.0"

No data is stored, hence no database is used. Default database sqlite is present.

Steps to setup the application:-

=>fetch the source from git
=>execute 'bundle:install' to get the gems installed
=>execute 'rails c' to access the console
=>execute 'rails s' to access the UI of the application from the browser at "http://localhost:3000"

Application is ready to be tested from the console as well as from the browser.
